# Install
Pour installer les modules, rendez-vous dans le dossier racine du projet puis :
```
    npm install
```

# Run server
Dans le dossier racine du projet, vous pouvez démarrer le serveur sur le port
3000 en faisant :
```
    npm run start
```
# http://localhost:3000/
Plus qu'à créer des équipes exotiques.
